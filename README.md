# dotfiles

Backup configuration

## Shell 脚本

| name         | description                                          |
| ------------ | ---------------------------------------------------- |
| install.sh   | Backup origin config then copy dotfiles to ~/.config |
| uninstall.sh | remove dotfiles then copy origin config to ~/.config |
